环境搭建

准备环境

Node.js、Hexo、NexT、git、github帐号
Node为Hexo提供环境支持，NexT，你可以选择其他的，我比较喜欢简洁的事物。
github提供了300M的域名空间，本教程也是发布到github上，免去了去购买服务器和搭建的麻烦事。
git是将Hexo编译后的文件推送到github上的工具。
好了，讲了这么多该动起你的双手了，GO->

Hexo安装

同样Easy的操作，执行以下指令，即可完成Hexo的安装。

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
hexo d -g //此指令用于打包上传至github,后面会讲到
